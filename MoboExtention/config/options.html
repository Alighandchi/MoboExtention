Getting Started

Follow these steps to set up and use the MoboVPN proxy service.

How to Start the Service

To turn on the proxy service, follow these instructions:

  1. Open the folder where the MoboVPN extension is located.
  2. Double-click the file named Manage Service.vbs.
  3. A pop-up window will appear asking you to confirm. Click Yes.
  4. Another window will ask for your Token ID. Enter your token and click OK.
  5. The Proxy service will start automatically in the background, and you can now use the extension.

How to Stop the Service

To turn off the proxy service, follow these instructions:

  1. Open the same folder where the MoboVPN extension is located.
  2. Double-click the file named Manage Service.vbs again.
  3. When the pop-up window appears asking you to confirm, click No.
  4. The Proxy service will be stopped automatically.

Getting Started

How do I add a new proxy?
You can add proxies manually using the “Add New Proxy” button. Enter each field separately or paste a full proxy string in one of these formats:
  • protocol://ip:port
  • protocol://user:pass@ip:port
  • protocol://ip:port:user:pass
Can I import and export proxies?
Yes! Use the Import and Export buttons on the Manage Proxies page to back up or transfer your proxies. Data is saved as a JSON file.

Proxy Behavior

What happens when I close the browser?
If the “Always connect to last used” setting is enabled, the extension will automatically reconnect to your last active proxy on startup.
Why does it say “Connected” when it's not working?
This can happen if the proxy credentials are no longer valid. The extension will now test the proxy before fully connecting and retry up to 5 times before marking it as disconnected.
How does reconnecting work?
When a connection fails, the extension will automatically retry up to 5 times (with a 5-second delay between attempts). Toast messages will show retry progress and result.
How do I disconnect a proxy?
Click the red Disconnect button in the popup. This will reset the proxy and clear authentication from cache and cookies.

Security & Storage

Where is my proxy data stored?
All proxy information is securely stored using Chrome’s storage.local API. Nothing is uploaded or synced externally.
Is authentication info safe?
Yes. Credentials are only used when establishing a proxy connection and are not stored beyond the session unless needed for reconnecting.
How do I clear proxy login prompts?
If Chrome gets stuck on a login popup, click Disconnect in the extension. This clears cached credentials and resets the session.
